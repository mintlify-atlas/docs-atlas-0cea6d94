Gova is a declarative GUI framework for Go. You define your UI as plain Go structs, compose views from layout primitives and widgets, and wire up reactive state with a single function call. The result is a native desktop window — no JavaScript runtime, no embedded browser, no extra assets to ship.

## What makes Gova different

Gova is designed around Go idioms, not ported from a web or mobile framework. Every public API uses plain structs, interfaces, and generics — no reflection magic, no string keys, no hidden scheduler.

## A minimal app

```go main.go
package main

import g "github.com/nv404/gova"

type Counter struct{}

func (Counter) Body(s *g.Scope) g.View {
    count := g.State(s, 0)
    return g.VStack(
        g.Text(count.Format("Count: %d")).Font(g.Title),
        g.HStack(
            g.Button("-", func() { count.Set(count.Get() - 1) }),
            g.Button("+", func() { count.Set(count.Get() + 1) }),
        ).Spacing(g.SpaceMD),
    ).Padding(g.SpaceLG)
}

func main() {
    g.Run("Counter", g.Component(Counter{}))
}
```

<Steps>
  <Step title="Install Gova">
    ```bash
    go get github.com/nv404/gova@latest
    ```
  </Step>
  <Step title="Write your first component">
    Implement the `Viewable` interface by adding a `Body(s *gova.Scope) gova.View` method to any struct.
  </Step>
  <Step title="Run the app">
    ```bash
    go run .
    ```
    Or use the CLI for hot reload: `gova dev .`
  </Step>
</Steps>
